## 🌟 Heap

### 1️⃣ Heap이란?

- 항상 **최소값(기본)** 또는 **최대값**을 빠르게 꺼낼 수 있는 자료구조
- 파이썬 `heapq`는 **최소 힙(min-heap)** 구조

---

### 2️⃣ 사용해야 하는 상황

- 최소값 또는 최대값을 **여러 번 반복해서 꺼내야 할 때**
- 값이 **계속 추가되면서도 우선순위를 유지해야 할 때**
- 입력 크기 N이 큰데(예: 10⁵ 이상) 최소/최대 연산이 반복될 때

---

### 3️⃣ 기본 문법

- `import heapq`
- `heapq.heappush(list, x)` → 값 삽입
- `heapq.heappop(list)` → 최소값 제거 및 반환
- `heap[0]` → 최소값 조회 (제거하지 않음)

---

### 4️⃣ 최대 힙 구현 방법

- 삽입 시 음수로 변환하여 저장
- 꺼낼 때 다시 음수 부호를 제거

---

### 5️⃣ 시간 복잡도

- 삽입 → O(log N)
- 제거 → O(log N)
- 반복 사용 시 전체 → **O(N log N)**

---

### 6️⃣ sort와 비교

- 한 번만 정렬하면 될 경우 → `sort()`
- 최소/최대 추출이 반복될 경우 → `heapq`
- 삽입과 추출이 계속 반복될 경우 → `heapq`
