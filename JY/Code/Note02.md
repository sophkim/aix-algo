## 🔗 str.join() (문자열 합치기)

### 🍎 str.join()

- **문법**: `"구분자".join(iterable)`
- **의미**: `iterable` 안의 **문자열 요소들**을 `"구분자"`로 **사이에 끼워 넣어** 하나의 문자열로 합침
  - 결과는 **항상 문자열(str)**
  - `iterable`은 리스트/튜플/문자열 등 **반복 가능한 객체**

**예시**

#### 1) 리스트 합치기

- `", ".join(["a", "b", "c"])` → `"a, b, c"`
- `"".join(["a", "b", "c"])` → `"abc"`

#### 2) 튜플 합치기

- `"-".join(("2026", "02", "15"))` → `"2026-02-15"`

#### 3) 문자열도 iterable (문자 단위로 쪼개짐)

- `".".join("ABC")` → `"A.B.C"`

**포인트**

- `join`의 주체는 **구분자 문자열**이고, 합칠 대상은 **iterable**이다.
  - 예: `" / ".join(list)` (O) / `list.join(" / ")` (X)

---

### ⚠️ 주의사항

#### 1) 요소는 **전부 문자열이어야 함**

- `", ".join(["a", 1, "b"])` → ❌ `TypeError`

- `", ".join(map(str, ["a", 1, "b"]))` → `"a, 1, b"`

#### 2) `join`은 **구분자가 앞뒤에 붙는 게 아니라 "사이"에만 들어감**

- `"|".join(["a", "b", "c"])` → `"a|b|c"`
  - `"|a|b|c|"` 이런 형태가 아님

#### 3) 빈 iterable이면 빈 문자열 반환

- `",".join([])` → `""`
