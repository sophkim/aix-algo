## 📚 구현 (Implementation)이란?

**머릿속에 있는 알고리즘을 소스 코드로 정확하게 옮기는 과정**을 의미한다.

### 특징

- 알고리즘 자체보다는 **문제 조건을 정확히 이해하는 것**이 중요하다.
- 사소한 조건 누락이나 구현 실수로 오답이 발생하기 쉽다.
- 문제에서 요구한 동작을 **그대로 코드로 표현하는 능력**이 필요하다.

### 대표적인 유형

1. **완전 탐색 (Brute Force)**

   - 모든 경우의 수를 빠짐없이 계산하여 해결하는 방법
   - 경우의 수가 많을 경우 시간 복잡도에 주의해야 한다.

2. **시뮬레이션 (Simulation)**
   - 문제에서 제시한 알고리즘이나 규칙을 **한 단계씩 차례대로 직접 수행**하는 방법
   - 구현 과정에서 조건 처리와 순서가 매우 중요하다
