## 📚 다이나믹 프로그래밍(Dynamic Programming)이란?

- 큰 문제를 작게 나누고, 같은 문제라면 한 번씩만 풀어 문제를 효율적으로 해결하는 기법이다.

### 1. **Top-Down 방식 (메모이제이션, Memoization)**

- 재귀를 사용하여 문제를 해결한다.
- 이미 계산한 결과는 **리스트에 저장**하여 다시 계산하지 않는다.

### 2. **Bottom-Up 방식**

- 작은 문제부터 차례대로 계산한다.
- **반복문**을 사용하여 DP 테이블을 채운다.
- 이 방식을 **주로** 사용한다.
