## 📚 정렬(Sorting)이란?

### 1. 순차 탐색(Sequential Search)

- **정렬되지 않은 리스트**에서 데이터를 찾을 때 사용한다.
- 리스트 안의 **특정 데이터를 찾기 위해 앞에서부터 하나씩 차례대로 확인**하는 방법이다.
- 데이터가 리스트의 끝에 있거나 존재하지 않는 경우, **모든 데이터를 확인**해야 한다.
- **시간 복잡도:** 최악의 경우 **O(N)**

---

### 2. 이진 탐색(Binary Search)

- **배열 내부의 데이터가 정렬되어 있을 때만** 사용할 수 있다.
- 탐색 범위를 **절반씩 줄여가며** 데이터를 탐색하는 방식이다.
- 이진 탐색은 **3개의 위치 변수**를 사용한다:
  - **시작점(start)**
  - **끝점(end)**
  - **중간점(mid)**
- 중간점이 실수일 경우, **소수점 이하는 버린다**.
- 찾으려는 데이터와 **중간점에 위치한 데이터**를 반복적으로 비교한다.
  - ( 찾으려는 데이터 < 중간값 ) → **끝점 = 중간점 - 1**
  - ( 찾으려는 데이터 > 중간값 ) → **시작점 = 중간점 + 1**
- **시간 복잡도:** **O(log N)**
- **구현 방법:** (1) 재귀 함수 (2) 반복문
